### **1\. Technical Stack**

* **3D Engine:** Three.js (Version r178 used).  
* **Smooth Scrolling:** Lenis.  
* **Page Transitions:** Barba.js.  
* **Styling:** Custom CSS built on top of Bootstrap (implied by \--bs- variables).  
* **Font:** IBM Plex Mono (Weights: 300, 400).

### **2\. Core Assets (Copyable Links)**

### **5\. Cloning Strategy**

* **Environment:** Set up a Vite or Webpack project.  
* **HTML:** Copy the structure inside \<body\> (Header, Scroller, Footer, Canvas).  
* **Styles:** Import the bundle.css or copy the core variable overrides.  
* **Three.js:** You will need to write the initialization code for the .glb model. The bundle.js contains the custom logic for rotating the camera/model based on scroll position (linked via Lenis).
